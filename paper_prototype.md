### Color Coding
- **Red (#DC2626)**: Primary brand, blood type badges, critical status
- **Blue (#2563EB)**: Secondary actions, links
- **Green (#16A34A)**: Success states, sufficient stock
- **Yellow (#F59E0B)**: Warning states, low stock
- **Gray**: Text, borders, backgrounds

### Typography
- **Headers**: Bold, larger size (Poppins)
- **Body**: Regular weight (Inter)
- **Labels**: Medium weight, smaller size

### Spacing
- Cards: 1.5rem padding
- Grid gaps: 1.5rem
- Modal padding: 2rem
- Button padding: 0.75rem Ã— 1.5rem

### Interactions
1. **Hover states**: Slight elevation, color change
2. **Active states**: Scale transform, gradient background
3. **Transitions**: 0.2-0.3s ease for smooth animations
4. **Loading states**: Disabled buttons, processing text

### Accessibility
- High contrast text
- Clear focus indicators
- Keyboard navigation support
- Screen reader friendly labels
